📋 GA4 연동 설정 방법

  1. analytics.google.com에서 GA4 속성 생성 → Measurement ID(G-XXXXXXXX) 확인
  2. index.html 상단의 G-XXXXXXXXXX를 실제 ID로 교체 (2곳)
  3. script.google.com → 새 프로젝트 → scripts/ga4-proxy.gs 코드 붙여넣기
  4. Apps Script에서 PROPERTY_ID를 GA4 속성 숫자 ID로 교체
  5. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자 → 배포
  6. 배포 URL을 이 파일 상단 APPS_SCRIPT_URL에 붙여넣기
